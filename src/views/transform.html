Transform

The CSS transform property lets you modify the coordinate space of the CSS visual formatting model. Using it, elements can be translated, rotated, scaled, and skewed according to the values set.






.transform { width: 100%; .outerbox { height: 400px; width: 80%; margin: 0 auto; background-color: aliceblue; } .innerbox { height: 50%; width: 30%; background-color: tomato; transform: rotate(30deg); } }

Options

scale

Enlarges an element on its X and Y axis.

transform: scale(2,3)

scaleX

Enlarges an element alone its X axis.

transform: scaleX(3)

scaleY

Enlarges an element alone its X axis.

transform: scaleY(3)

skewX

Skews the element along the X axis by the given angle.

transform: skewX(-30deg)

skewY

Skews the element along the Y axis by the given angle.

transform: skewY(-30deg)
